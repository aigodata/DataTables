前言

自我介绍

我们一贯的风格就是拒绝重复发明轮子, 选型DataTables作为ag-ui的表格补充, 我们在它的基础上优化了样式.

$("#unfold-row-wrap").on('click', function (evt) {
  // 除了展开收齐图标, 其余的点击进行remove
  if(!$(evt.target).hasClass("detail-control")) {
    var tr = $(this).find('tr');
    tr.removeClass('detail-bar');
  }
});
$("#unfold-row").on('click', ".detail-control", function (evt) {
